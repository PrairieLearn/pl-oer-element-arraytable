This question is used to demonstrate how weights of different elements impact the grade received in the question. The first question below has the weight attribute set to 2, while the second question has a default weight of 1. So, if the first question is correct, and the second is 100% wrong, the student will receive 66%. If the second question is correct and the first question is 100% wrong, the student will receive 33%.
