The following questions use the attributes data-base, data-prefix, signed, and size.

data-base is used to specify what kind of input the student is supposed to enter in the table (aside from unknown values). This information is used when parsing to make sure they input the right type of data as well as in grading when comparing the submitted answer to the correct answer. Its default value is "dec". Note that for all of these, adding a prefix such as "0b" to the front of an answer will not change the resulting grade.

Part 1: Hexadecimal Value

In this first question, data-base is set to "hex". So certain letters are allowed while others, such as "s", are not. This is a signed value.
