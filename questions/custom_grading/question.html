This question demonstrates how you can make a custom grader for the array-input element. In this question, the important indices that need to be modified are 1, 2, and 3, so if everything except those is correct, then we would want the grade to be lower than 72% (8/11), as the most important requirements are not met. This is handled via a custom grading function in server.py.

In this question, the three relevant indices are given 75% weight.

Say we make the following changes to the array below, A, in order. What would the resulting A look like?

A[1] = 10;

A[2] = 5;

A[3] = 6;
